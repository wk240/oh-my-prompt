# 快速开始

5分钟上手 Oh My Prompt。

## 前置条件

- Chrome / Edge 浏览器（Chromium系）

## 安装

### 下载安装包

1. [下载最新版本](https://github.com/wk240/oh-my-prompt/releases)
2. 解压 zip 文件
3. Chrome 打开 `chrome://extensions/`
4. 启用「开发者模式」
5. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 基础使用

1. **一键插入提示词**
   - 在 Lovart 等 AI 平台的输入框旁点击 ⚡ 图标
   - 选择提示词 → 自动插入

2. **图片转提示词**（需配置 API）
   - 鼠标悬停网页任意图片 → 点击 ✨ 按钮
   - 自动生成双语提示词，并保存到临时库

3. **提示词Agent**（需配置 API）
   - 点击浏览器工具栏图标 → 打开 Agent
   - 简短描述自动生成详细提示词

4. **团队共享**
   - 点击浏览器工具栏图标 → 打开团队库
   - 选择提示词 → 自动插入
