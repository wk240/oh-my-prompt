## Color

Based on shadcn neutral preset with CSS variables.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--background` (white) | Popup surface, card backgrounds |
| Secondary (30%) | `--muted` (light gray #f5f5f5) | Category sidebar, hover states, separators |
| Accent (10%) | `--primary` (near-black #171717) | Primary CTA button, selected category indicator, focus ring |
| Destructive | `--destructive` (red #ef4444) | Delete buttons only |

**Accent reserved for:**
- Primary CTA button ("添加提示词")
- Selected/active category highlight
- Focus ring on interactive elements
- Import/Export action icons when active

## Layout Constraints

- **Popup width:** 300px (Chrome extension standard)
- **Popup height:** Auto (max ~600px with scrolling)
- **Category sidebar:** ~80px width, full height
- **Prompt editor area:** Remaining width (~220px)

---

## Visual Hierarchy

**Primary focal points (by state):**

| State | Focal Point | Reason |
|-------|-------------|--------|
| Empty | Primary CTA button ("添加提示词") | Directs user to first action |
| Populated | Prompt list/content area | Primary content users interact with |

**Visual hierarchy order:**
1. Category sidebar (navigation anchor)
2. Prompt list/content area (primary content)
3. Primary CTA button (action trigger)
4. Import/Export actions (secondary actions)
